从C到C++(3)："万能"的vector

本文将介绍c++中常用的容器类，vector

1.容器与C++中的容器类

容器是什么呢，这很容易理解，是装东西的东西，换句话说，这个东西里边有其他的东西，我们就把它叫做容器，编程中的容器同样是如此。我们最常见到的容器就是数组，比如，一个 int 型的数组便是一堆 int 数据的容器，C风格的字符串也是很常见的容器，是字符的容器。

那么，既然数组这么万能，为什么我们需要容器这个东西呢。这要从数组的缺点说起了

在实际的编程中，我们经常会遇到这样的问题：

  • 问题1：数组不能自动扩容，而我们很多时候不知道要处理数据的数量，这是最经常性的一个问题了。

  • 问题2：数组不能作为程序的返回值,在C/C++语言中。而我们会经常性的需要从函数返回一个数组。但特殊的，char * 的C风格字符串可以作为直接的返回值

    这些情况下，数组就并不是一种很好的选择。C++在标准库中，定义了一批容器类来解决这种问题，在本文中，我们只介绍用的最多，除非在有特别需求的地方，否则任何时候你都可以用的一个具有代表性的容器类，“万能”的 vector 类

最后一种定义方法，和 JAVA 中对 ArrayList 的操作不同，使用这种方法定义一个 ArrayList 时，两个变量将指向同一个地址，相反，在C++中，两个变量将指向不同的地址。
