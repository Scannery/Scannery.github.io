2.C++中的标准输入和输出

cin 与 cout 函数存在于头文件 iostream 中

这里说的输入输出只包括控制台的输出

std::cin>>s>>s1>>s2>>sn;

上式为标准输入，将控制台输入的值置入s中,s可以为整型，长整，浮点，字符串……

在标注了 using namespace std; 的情况下，上式可以简化为:

cin>>s1>>s2>>sn;

注意：使用连续的输入时，可以用空格来分割两个输入，也可以一个一个输入，即输入一个数据接空格

比如，我用cin>>s1>>s2>>s3时，我可以这么输入:

s1\回车
s2 s3\回车

c++的标准输出如下：

std::cout<<输出1<<输出2<<输出n<<std::endl;

c++标准输出由cout与endl两部分组成，输出为任意变量或用双引号引起的字符串

同理，在 using 掉 std 的情况下，上式简化为:

cout<<输出1<<输出n<<endl;

其中， endl 是end line 的意思，表明输出之后将自动换行

值得注意的是：

不能用 cin 或 cout 直接输出非特殊类的对象

5.string 类的常用方法

s.empty()

返回一个布尔值，如果字符串s为空，返回 true ，否则返回 false

布尔值用 boolean 变量 定义,取值有 true 和 false ，在C语言中，我们用0表示 false ，用非0表示 true

关于更多布尔值的资料，请戳这里

s.size()

返回字符串的长度，注意，字符串的顺序是0到长度-1；

s[n]

返回字符串 s 在 n 位置的 char 的引用，大家可以将其理解为值，更多的引用得说明，请参照下篇

初学的朋友可以无视引用二字，理解为在n位置上的值就可以了

注意， s[s.size()] 越界了，最后一个字符是 s[s.size()-1]!!!

s1+s2

将 s1 与 s2 相连，如 s1 为 abc ，s2 为 bcd ，那么 s1+s2 的值为 abcbcd

s1=s2

将 s2 的值赋予 s1;

s.erase(起点,终点)
s.erase(位置)

第一种方式，擦除字符串内起点到终点位置的字符，起点终点可以为 int 型或迭代器 string::iterator 型

第二种方式，擦除字符串中指定位置的字符，起点终点可以为 int 型或迭代器 string::iterator 型

s.push_back(字符)

在字符串尾部加入一个字符

你也可以写成 s+字符 来达成这个效果

s.c_str()
s.data()

返回一个C风格的字符串，或者说成字符数组，用 c_str 将会有 \0 的结束符，data 则没有 \0 的结束符
