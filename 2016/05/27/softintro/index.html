Scannery 的软件推荐：windows篇

本文将推荐一些个人用的习惯的软件

博客大致会做到周更，闲时可能会多更

Windows 软件

bililocal 视频播放器

bililocal 视频播放器是一个 本地 的弹幕播放器，他可以让你在看本地视频的同时同样可以享受到网络上的弹幕

下载地址或想了解更多: 软件更新帖

源码地址: github源码

本作还有linux版

界面如下:

bililocal 同时支持 Acfun 的弹幕

值得一提的是，博主的版本加载字幕时候会特别卡，可能最新版已经修复了这个问题

另外安利腾讯的QQ播放器，算是一个功能较为齐全，且适合国人使用习惯的播放器了，腾讯难得的良心软件

chrome 谷歌浏览器

谷歌是一个值得尊敬的公司，作为一个国际级的公司，他的关注点在 前景 ，而非国内大部分公司所关心的 钱景 上

我们熟悉的手机操作的系统 Android ,便由谷歌公司参与开发

谷歌浏览器是一款非常好用的浏览器，并且有功能丰富且使用的小插件，比如:

  • 广告屏蔽插件 : Adguard

  • 翻译插件 : Google Translate

  • 键盘操作插件，用键盘浏览网页 : Vimium

  • 更多知乎大大推荐的好用插件，戳这里

Lantern

蓝灯是一款非常简单的翻墙软件，基本安装上之后，让他开机启动，就可以直接去墙外了。比较省事的一款软件

值得一提的是，使用代理也可以加快访问非被墙网站的速度，比如git,twitch之类
