第二步.安装与配置hexo

1.git shell简介

git shell可以使你在电脑上运行shell语言，换而言之，你可以在这个窗口上运行部分linux系统上的命令行。你不懂也没关系，这里给出了一些常用的shell命令:

最后提醒一遍，不知道说啥的朋友请先看这篇文档

  • cd [路径名]:进入指定路径. 如 cd c:\Users\temp 进入c\用户\temp的路径

    路径是什么呢？就是这个东西，表示控制终端工作的地方，如图：

    特殊的，当前路径可以用 . 来表示，比如 cd . 就是进入当前文件夹,也就是没动……上一层的路径可以用 .. 来表示， cd .. 就是进入上层路径

  • touch [文件名]:创建指定文件.如 touch cac.txt 创建一个名为cac的txt文件

  • cat [文件名]:将指定文件打印到控制台

  • rm [文件名]:删除指定文件

  • vim [文件名]:用vim文本编辑器打开文件，熟悉这个命令的朋友可以用vim来编辑下边谈到的文件，不熟悉的朋友建议别碰

    注意，以上命令冒号后边一定要加个空格

第四步.使用markdown书写一篇博文

markdown是一种文本编写方式，可以实现书写+排版同时进行，节省排版的时间，详细参考百度百科

1.新建一个markdown文件

+在sublime里，寻找到hexo文件夹里的source文件夹，进入其中找到_posts，打开文件夹。

+点击左上角file,new file,创建一个新的文件，在sublime右下方找到plain text标签，左键，在列表中找到Markdown，点击并选取里边的markdown选项

+按Ctrl+s，选择到hexo\source_post文件夹下，并保存。

+在新建的文件上点击右键rename,将其改为md后缀名
