5.迭代举例，快速排序算法

我们举一个快速排序的例子，来看看这种经典排序方法的执行步骤。

1.快速排序的原理，

快速排序的本质是将数组一份为二。

首先，我们再数组中取一个 key 值，我们进行排序交换，将小于 key 的值放到 key 的左边，将大于 key 的值放在 key 的右边。

之后，我们将 key 左边，右边的数组进行同样方式的排序。再次分割，然后将分割后的四部分进行再次的快速排序……

直到，划分的每个数组元素量为1，或者，划分后的数组排序并没有经历过一个数据移动，即为数组子集的完成

值得一提的是，快速排序的时间复杂度为 n(log2(n)) ;相比于冒泡排序的 n平方 要快不少。

2.简单的快速排序说明

我们定义一下数组:

3,5,6,2,7

取 key 为1，定义 i=0 ，j = 4（即数组的首和尾)

从尾开始排序，7>3，合理，2<3，交换2和3，数组变为

2,5,6,3,7

这时候，j 的值保存为 3，i 变量进行自增，2<3，但5>3 ，交换二者

2,3,6,5,7

这时候，i 的值保存，j再开始减，直到 i 的值和 j 的值相等时，我们发现

数组已经划分为 key 值3的左右两部分，左边小于三，右边大于三

我们将集合分为两部分，2，和6,5,7

2 6,5,7

2为单字，排序完成，进行6,5,7排序，

同上，取第一个数字 6 为 key 值。 i 设为起始值2，j设为子数组最大元素4.同理进行排序
